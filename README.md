# Excalidraw Clone

A collaborative whiteboard application inspired by Excalidraw, built with React and TypeScript. This project is designed to be offline-first and evolves through multiple phases from a frontend-only V1 to a real-time collaborative V4.

## Features

- **Offline-First**: Works seamlessly without internet connection
- **Canvas-Based Drawing**: High-performance canvas rendering
- **React + TypeScript**: Modern, type-safe development
- **Offline Persistence**: Local data storage with RxDB
- **Collaborative Ready**: Architecture prepared for real-time collaboration

## Project Phases

- **V1**: Frontend-only, offline-first whiteboard
- **V2**: Cloud sync capabilities
- **V3**: Multi-board support
- **V4**: Real-time collaboration

## Tech Stack

- React
- TypeScript
- RxDB (offline database)
- Canvas API
- pnpm workspaces
- Biome (linting & formatting)

## Getting Started

### Prerequisites

- Node.js
- pnpm

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd excalidraw-clone

# Install dependencies
pnpm install
```

### Development

```bash
# Run linting
pnpm lint

# Format code
pnpm format

# Check code
pnpm check
```
